# Экраны и фичи

> Поэкранная спека. Макеты — в `docs/design/` (файлы «… · Стиль 3.html»). Статусы доступа отражают текущую бизнес-модель (см. раздел «Доступ» ниже).

## Навигация

Нижний таб-бар, 3 раздела: **Главная** · **Функции** · **Профиль**.
Отдельные флоу (чек-ин, гороскоп, недоплачивают, совместимость, подписка) открываются поверх и имеют кнопку «назад»/«закрыть».

## Доступ: free / paid

**Бесплатно (делаем сейчас):**

1. **Ежедневный чек-ин** + отчёты (недельный, месячный) + просмотр своих метрик каждый день. _(Идея: чек-ин бесплатен ~первую неделю — точную gating-логику уточним позже.)_
2. **«Сколько недоплачивают»** — ограниченное число раз в месяц. Вкладка результата **«В других городах лучше» — только PRO**.
3. **Совместимость** с боссом/коллегой/бывшим.
4. **Офисный гороскоп**.

**Платно (НЕ реализуем сейчас, только пейволл-модалка):** 5. **«Важный разговор»** — подготовка разговора с боссом с ИИ. 6. **Еженедельный отчёт о рынке** — расширенный авто-мониторинг (по сути расширенная «Недоплачивают», раз в неделю, автоматически).

**Монетизация сейчас:** реальной оплаты нет. Тап по PRO → bottom-sheet/модалка «Скоро будет готово, ты первый узнаешь». Замеряем намерение (клики «Попробовать/Оплатить»).

## 1. Главная (`Главный экран · Стиль 3.html`) — free

- Акцентный hero: «Привет, {Имя}», аватар, маскот, кнопка **«Пройти чек-ин»**.
- Ряд метрик (наезжает на hero): **Сон** (напр. 6.5ч), **Сост.** (85%), **Выгор.** (15%), **Отдых** (40%).
- Секция «Инструменты» — плитки 2×2: Сколько недоплачивают · Офисный гороскоп · Совместимость · Важный разговор (PRO).
- Нижняя навигация (Главная активна).

## 2. Функции (`Функции · Стиль 3.html`) — free

- Топбар «Функции» + иконка поиска.
- Сетка плиток: Сколько недоплачивают, Офисный гороскоп, Совместимость, **Важный разговор (PRO)**, **Автопоиск вакансий (PRO)**, **Недельный отчёт (PRO)**.

### 2a. Апселл-шит (`Функции Апселл · Стиль 3.html`) — пейволл

- Тап по PRO-плитке → bottom-sheet поверх затемнения.
- Бейдж «Premium», иконка, заголовок фичи, описание, список перков (3 пункта), кнопка «Открыть Premium · 199 ₽/мес», «Восстановить».
- **Сейчас:** вместо реальной оплаты — сообщение «Скоро будет готово».

## 3. Профиль (`Профиль · Стиль 3.html`) — free

- Hero-аватар с камерой, имя + редактирование.
- Блок «Личное»: дата рождения, время рождения (опц., «для точности»).
- Блок «Работа»: страна, город, профессия, должность/грейд.
- (В макете есть upsell-карточка `.upg` — апселл Premium.)

> Профиль — источник данных для персонализации фич (недоплачивают, гороскоп, совместимость).

## 4. Чек-ин (ежедневный ритуал) — free

Флоу из шагов с прогресс-баром (`2/5`), маскот, вопрос, шкала.

- **Вариант 1–5** (`Чек-ин · Стиль 3.html`): сетка 5 квадратных кнопок, полюса «вымотан → в потоке».
- **Вариант 1–10** (`Чек-ин 1-10 · Стиль 3.html`): два ряда по 5, полюса сверху/снизу.
- Метрики чек-ина (по экрану «Готово»): **настроение, энергия, сон, стресс, опора** (5 вопросов).
- Кнопки: назад + «Дальше».

> Длину шкалы задаёт сам вопрос (`question.answers` — массив значений). По умолчанию **1–5**;
> компонент `CheckInScale` рендерит и 1–10 (5 колонок в 2 ряда, полюса сверху/снизу).
> Вопросы приходят запросом `getCheckInQuestions` (`id`/`order`/`title`/`helperText`/`lowText`/`highText`/`image`/`answers`),
> где `id` совпадает с ключом метрики (mood/energy/sleep/stress/support).

### 4a. Чек-ин Готово (`Чек-ин Готово · Стиль 3.html`)

- Hero «Чек-ин записан» + поддерживающий заголовок + дата/время.
- `.snap`: 5 метрик /5 (настр./энергия/сон/стресс/опора).
- Карточка «Полезный совет» (контекстная, на основе ответов).
- Кнопки: «Посмотреть динамику», «На главную».

## 5. Сколько недоплачивают — free (с лимитом)

### 5a. Форма (`Недоплачивают Форма · Стиль 3.html`)

- Поля: Страна, Город, Профессия, Грейд (чипы), Текущая зарплата (опц.).
- Подпись «Данные не уходят на сервер».
- Кнопка «Показать вилку».

### 5b. Результат (`Сколько недоплачивают · Стиль 3.html`)

- Акцентный топ: «Senior · Frontend · Москва».
- Карточка медианы: **350 000 ₽**, вилка 280–450к, гистограмма распределения с маркером «ты», строка «Ты 220к — на 60к ниже минимума».
- Табы: **«Крутые вакансии»** (free) · **«В других городах лучше»** (PRO).
- Список вакансий (лого, роль, компания, источник HH, зарплата, diff «+130к ↗»).
- Кнопка «Сохранить отчёт».

## 6. Офисный гороскоп — free

### 6a. Форма (`Гороскоп Форма · Стиль 3.html`)

- Заголовок, маскот-астролог, поле даты рождения, мини-карточка определённого знака («Дева · земля»).
- Кнопка «Узнать расклад на сегодня».

### 6b. Расклад (`Гороскоп · Стиль 3.html`)

- Hero со знаком (Дева, даты, стихия).
- Плашка «Меркурий ретроградит».
- Лид-текст (самоироничный прогноз на день).
- Строки-аспекты: Созвоны (риск ↑), Дедлайны (терпимо), Общий чат (опасно) — с цветными пилюлями.
- Кнопки: «Поделиться раскладом», «На главную».

## 7. Совместимость — free

### 7a. Форма (`Совместимость Форма · Стиль 3.html`)

- Твоя дата рождения (+ автоопределённый знак).
- Сегмент «Кто это?»: Босс / Коллега / Бывший.
- Дата рождения второго (+ знак).
- Кнопка «Проверить совместимость».

### 7b. Результат (`Совместимость · Стиль 3.html`)

- Акцентный топ: пара знаков (ты × босс), большой **%** (34%), вердикт («Выживаете на выносливости»).
- Шкала «катастрофа → идеал» с маркером.
- Описание-текст.
- 3 бублика: Дедлайны / Фидбэк / Терпимость.
- Кнопки: «Поделиться отчётом», «На главную».

## 8. Подписка (`Подписка · Стиль 3.html`) — пейволл

- Полноэкранный bottom-sheet поверх затемнённого фона.
- Акцентный хедер: корона, «Держимся Premium».
- Планы: **Месяц 199 ₽** / **Год 124 ₽/мес (1 490 ₽, −38%)**.
- Список «Что откроется»: Важный разговор, полный доступ к «Недоплачивают», недельные отчёты + автопоиск, аналитика выгорания.
- CTA «Попробовать 7 дней бесплатно».
- **Сейчас:** оплаты нет → «Скоро будет готово, ты первый узнаешь» + замер намерения.
